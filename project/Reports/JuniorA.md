## How did you communicate with your team lead? Was there any issues?
The lead developer set up a discord server for the project they led. 
There were no issues in communications, any questions we had could just
be asked in the chat and if the lead developer did not have an immediate 
answer, they would get back to us when they did.

## Did you know what was expected of you for each site?
Sometimes there was a bit of confusion, but some questioning
led to the quesitons being resolved pretty quickly so I could
know what was expected or assigned to me.

## Were you able to contribute to your sites?
I was able to contribute to the html and CSS of the sites
I was working on. On this site I contributed to the html
for the price-graph.html file and price-graph.css file.

## Did you run into any issues with GitHub and adding your changes to the sites?
I occasionally ran into a bug with GitHub where the CSS would not always update
properly. This usually seemed to happen when I would push an update that had changes
to CSS and html but it was not always consistent. Usually I could fix it by pushing a
change with only CSS modifications.
