## How did you communicate with your team lead? Was there any issues?
The lead developer set up a discord server for the project they led. 
There were no issues in communications, any questions we had could just
be asked in the chat and if the lead developer did not have an immediate 
answer, they would get back to us when they did. For example, I had a 
question about the CSS for price-graph and the lead dev said to use
the CSS from the home page as a template.

## Did you know what was expected of you for each site?
For the most part it was pretty straight forward. Like 
most of the projects, each person was assigned to a 
specific page, which means they were in charge of the
html and CSS for their page. I was assigned the
price-graph page. However, one difficulty I 
encountered was the initial wireframe I was using
provided from the canvas discussion was outdated.
I was not made aware until I asked since I could
not initially find the wireframe for price-graph.
However, after asking about the missing wireframe,
I was informed by the lead dev that there was an
updated site plan and after that it was sorted out.

## Were you able to contribute to your sites?
I was able to contribute to the html and CSS of the sites
I was working on. On this site I contributed to the html
for the price-graph.html file and price-graph.css file.

## Did you run into any issues with GitHub and adding your changes to the sites?
I occasionally ran into a bug with GitHub where the CSS would not always update
properly. This usually seemed to happen when I would push an update that had changes
to CSS and html but it was not always consistent. I found I could fix it by pushing a
change with only CSS modifications in most cases.
